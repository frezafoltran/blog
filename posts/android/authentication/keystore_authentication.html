Why sign the Android app?

The app needs to be signed for security reasons. For example, the signature ensures that updates to an existing app are only rolled out by the original developers.

How to sign the app

You must create a signing certificate and point the build.gradle files in your Android app to use the certificate you generated.

How to create a signing certificate

One way to do it is to use the keytool included in the Java SDK. The easiest way to use the tool is through Visual Studio. First, make sure to install the Xamarin tools. Then choose Tools -> Android -> Android Adb Command Prompt. Now you will have a command prompt with the JDK path included in the global paths.

To generate your keystore signing certificate run the command:

keytool -genkeypair -v -keystore (keystore file name).keystore -alias (your key alias) -keyalg RSA -keysize 2048 -validity 10000

The parameters can be modified and the a description of the information that will be requested can be found at this article.
